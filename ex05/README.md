# 第5回
## 負けるな!こうかとん(ex05/fight_kokaton.py)
### ゲーム概要
- ex05/fight_kokaton.pyを実行すると、1600x900のスクリーンに草原が描画され、こうかとんを移動させ飛び回る爆弾から逃げるゲーム
- こうかとんが爆弾と接触するとゲームオーバーで終了する
- ゲームオーバーになり次第、デフォルトブラウザで学内ポータルサイトが起動する。また、CSVファイルとExcelファイルの両スコアブックが生成される
- 写真アプリで使用したこうかとんを確認
### 操作方法
- 矢印キーでこうかとんを上下左右に移動する
### 追加機能
- Excelにおけるスコアブックを出力
- CSVにおけるスコアブックを出力
- 爆弾をランダムに1～6個の数に設定（単数個または複数個）
- こうかとんの様子をランダムに設定（例:困っているこうかとん、楽しんでいるこうかとん）
- 爆弾の色をランダムに設定
- 学内ポータルサイトを起動
- 使用したこうかとんを写真アプリで確認
